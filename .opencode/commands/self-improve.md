Use this self improve heartbeat workflow when prompt follow this format: 

> Self Improve Heartbeat [datetime-now-in-YYYYMMDD_hhss-format]
>
> E.g. Self Improve Heartbeat 20260328_1637

1. Go through [session journals](/journals/session/) that has yaml frontformater with `reflect: false` property

2. For each of the session journal, perform self reflection using gibbs-reflective-cycle skill 

   - describe: what happened?

   - feeling: what was i thinking at the time?

   - evaluation: what was good and bad about the interaction?

   - analysis: why did things go the way they did?

   - conclusion: what else i could have done?

   - action plan: 
   
     if master make the same request again, what can i do to increase master satisfaction?

     - create a skill to repeat the exact same process?
     
     - if skill already created, how can the process be even leaner, faster and consume lesser steps to achieve the same request?

     - are there tools/mcps out there that can make process easier?
     
     - should i write any automation script for the process?

   Write down your findings into your [notes](/notes/)
